# 🚴 Bike Sharing Demand Prediction

This project focuses on predicting bike rental demand using machine learning techniques. It leverages historical data such as weather conditions, seasonality, and time-based features to build an accurate regression model.

---

## 📌 Problem Statement

Bike-sharing systems generate large amounts of data related to rentals. The goal is to predict the number of bikes rented at a given time to optimize availability and improve operational efficiency.

---

## 📊 Dataset

The dataset typically includes:
- Date & Time
- Temperature
- Humidity
- Wind Speed
- Season / Weather conditions
- Count of bike rentals (target variable)

---

## ⚙️ Tech Stack

- Python
- NumPy
- Pandas
- Matplotlib / Seaborn
- Scikit-learn

---

## 🔍 Workflow

1. Data Loading & Cleaning  
2. Exploratory Data Analysis (EDA)  
3. Feature Engineering  
4. Model Training  
5. Evaluation  
6. Prediction  

---

## 🧠 Models Used

- Linear Regression  
- Decision Tree Regressor  
- Random Forest Regressor  

---

## 📈 Evaluation Metrics

- Mean Absolute Error (MAE)  
- Mean Squared Error (MSE)  
- R² Score
